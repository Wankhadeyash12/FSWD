FULL STACK E-COMMERCE WEB APPLICATION DOCUMENTATION(Amazon-like Platform with Razorpay & Cloudinary)1. Project OverviewProject Name: ShopSphereProject Type: Full Stack E-Commerce Web ApplicationObjective: Develop a scalable and secure online shopping platform with Razorpay payment integration.2. System ArchitectureClient (React) -> API Calls -> Backend (Node + Express) -> MongoDB -> Razorpay & Cloudinary6. Database DesignUsers, Products, Cart, Orders collections in MongoDB7. API Endpoints/api/auth/api/products/api/cart/api/orders/api/payment/api/upload8. Payment WorkflowCheckout -> Razorpay order -> Payment -> Verification -> Order success9. Image Upload WorkflowUpload -> Backend -> Cloudinary -> URL -> Store in DB10. DeploymentFrontend: VercelBackend: RenderDatabase: MongoDB Atlas11. SecurityJWT, bcrypt, HTTPS, validation, secure keys12. Development PhasesSetup -> Auth -> Products -> Cart -> Payment -> Cloudinary -> Admin -> Deploy13. DeliverablesComplete working app with admin panel, payments, and image uploads
